
📊 Анализ текста на Python
Этот проект на Python позволяет анализировать текст и выполнять разные операции со словами: подсчёт частоты, поиск слов, выделение и статистика.


 
 
🚀 Запуск программы
1. Установить Python
2. Скачать файл проекта
3. Запустить файл main.py в терминале или через PyCharm
4. Ввести текст
5. Выбрать нужное действие в меню
 



 
🧠 Возможности программы
После запуска пользователь вводит текст, а затем может выбрать один из вариантов:




🔍 1. Поиск слова
* Пользователь вводит слово
* Программа ищет все слова, где есть совпадение
* Показывает количество найденных слов
* Выводит найденные слова
* Показывает текст с выделением найденных слов (КАПСОМ)
 




 
🔥 2. Самые частые слова
* Программа анализирует текст
* Определяет, какие слова встречаются чаще всего
* Выводит ТОП-3 самых популярных слов
 



 
📊 3. Общее количество слов
* Программа считает все слова в тексте
* Показывает общее количество слов


 
 
🧩 4. Уникальные слова
* Программа убирает повторяющиеся слова
* Считает количество уникальных слов
* Выводит результат
 




⚙️ Как работает программа
1. Пользователь вводит текст
2. Программа приводит текст к нижнему регистру
3. Удаляются знаки препинания
4. Текст разбивается на список слов
5. Создаётся словарь с частотой слов
6. Пользователь выбирает действие из меню
7. Программа выполняет выбранную операцию
